# Release Notes v0.9.0

## Summary

v0.9.0 completes a docs-only provider suitability comparison across Sportmonks, API-Football/API-Sports, Capology and Transfermarkt as a manual reference.

The milestone records qualitative role outcomes without scoring, ranking, implementation or provider approval.

- Provider comparison matrix.

## Evidence And Governance

- Public official sources were prioritized.
- No login, API calls, cache reading, raw JSON or scraping.
- Unknowns, restrictions and conflicts remain documented.
- Numeric scoring was not used.
- No universal winner was selected.
- No multi-provider architecture was recommended.

## Limitations
