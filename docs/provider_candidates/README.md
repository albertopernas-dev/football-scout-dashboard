# Provider Candidates

This folder contains preliminary shortlists and review material for possible provider or source candidates.

No document here approves a provider, permits scraping or enables integration. Any payload inspection, credential setup or caching must follow the applicable gate conditions and complete the required checklist and decision record. Every real candidate must pass through:

- the [v0.7.0 Candidate Review Workflow](../v0_7_0_candidate_review_workflow.md);
- the [Candidate Review Pack Template](../v0_7_0_candidate_review_pack_template.md);
- the [Provider Payload Evaluation Checklist](../provider_payload_evaluation_checklist.md);
- a provider payload [decision record](../provider_decisions/provider_payload_evaluation_template.md).

Do not store real payloads, credentials, provider dumps or `.local.csv` files in this folder.
