# Demo Assets Checklist

Use this checklist to prepare screenshots, GIFs or a short video for GitHub, portfolio pages or LinkedIn.

## Screenshots To Capture

- [ ] App header / dataset summary.
- [ ] Data quality panel.
- [ ] Main player table with recommended ranking.
- [ ] Ranking filters visible.
- [ ] Opportunity Finder with filters.
- [ ] CSV download buttons.
- [ ] Similar players view.
- [ ] HTML scouting report example.
- [ ] Documentation links in README.

## Recommended Demo Video Flow

Target duration: 60-90 seconds.

1. Show README highlights.
2. Run or show the app already open.
3. Show dataset summary.
4. Filter reliable/comparable players.
5. Open Opportunity Finder.
6. Export a CSV shortlist.
7. Generate a scouting report.
8. End with limitations and next steps.

## Suggested Captions

- "SQLite-first football scouting dashboard"
- "Sample-adjusted rankings based on minutes reliability"
- "Opportunity Finder with reliability and comparability filters"
- "Built with Python, Streamlit, pandas, scikit-learn and Plotly"

## Do Not Claim

- [ ] Complete market intelligence.
- [ ] Real transfer recommendations.
- [ ] Real contract data.
- [ ] Real market values.
- [ ] Fully validated professional scouting model.
